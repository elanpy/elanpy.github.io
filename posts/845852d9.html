常用快捷键

动作 快捷键
快速全屏或取消全屏（需要软件支持） control + option + F
跳到文本开头/结尾（加 shift 可以选中） command + 左/右箭头
跳到上一次词头/词尾（加 shift 可以选中） option + 左/右箭头
反向搜索命令行历史记录 control + R
终端行清屏 command + K
终端清除到上一命令 command + L
左右切换桌面 control + 左/右箭头
新建窗口 command + N
新建标签页 command + T
